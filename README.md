
# 🤖 teachAi -

teachAi is an AI-powered conversational platform built using the MERN stack. This project provides real-time chat functionality by leveraging the OpenAI API (or any other Large Language Model).


---
🚀 Features
Real-time Chat: A fast, intuitive, and highly responsive chat interface.
MERN Stack: Robust architecture with Node.js/Express for the backend and React for the frontend.
Secure: Sensitive information like API keys are kept secure using Environment Variables (.env).
Code Highlighting: Integrated support for react-markdown to render programming code snippets beautifully.
Responsive Design: Fully optimized for all devices, including Mobile, Tablets, and Desktops.


## 🛠️ Tech Stack

**Frontend:**
*   React.js
*   Vite (Fast Build Tool)
*   CSS3 / Tailwind CSS
*   React Markdown

**Backend:**
*   Node.js
*   Express.js
*   OpenAI API
*   Dotenv
